# Full ERP System

A full ERP system designed for company operations, including HRIS, HR, warehouse, inventory, project management, job orders, service operations, and reporting modules.

## Overview

This project is a business management system that helps companies organize daily operations through structured modules and centralized workflows. It is built to support internal teams by managing employees, inventory, warehouse activity, service requests, projects, and operational records.

## Features

* HRIS and employee management
* HR and administrative workflows
* Warehouse management
* Inventory tracking
* Project management
* Job order management
* Service operations
* Reporting and records management
* Responsive web interface
